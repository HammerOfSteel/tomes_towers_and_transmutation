# Settlement Integration
> ⚠️ GAP — Place generated settlements from OW-A/B into the 3D overworld at correct realm map positions.

## Status: ⚠️ Not planned

## Goal
When the player walks toward a settlement dot on the realm map, they find actual 3D buildings there — matching the faction and layout generated by the settlement tool.

## Tasks

### SI-1 — Settlement Spawner
- [ ] `SettlementSpawner.ts`: given `SettlementRecord` (x, y, seed, faction, size), places building cluster at world position
- [ ] World position: `(x / realmW) * WORLD_SIZE, (y / realmH) * WORLD_SIZE`
- [ ] Uses `buildBuilding(dna)` from PROC-B for each ward's building type
- [ ] Spawns at terrain height (raycasts down from settlement centre to find ground)

### SI-2 — Street + Path Mesh
- [ ] Settlement roads from OW-A road network → `THREE.TubeGeometry` (or quad strip) at ground level
- [ ] Road width by type: main road 2 WU, alley 1 WU
- [ ] Road material: cobblestone canvas texture (settlement faction variant)

### SI-3 — NPC Population
- [ ] `SettlementPopulator.ts`: generates `NpcDNA[]` from ward types
  - Market ward → 2-3 merchants
  - Gate ward → 1-2 guards
  - Inn ward → 1 innkeeper + 1-2 wanderers
  - All → 1 quest-giver per town/city
- [ ] Calls `buildNpc(dna)` → spawns at random position within ward bounds
- [ ] NPCs have idle wander behaviour within ward radius

### SI-4 — Settlement Boundary
- [ ] Collision walls or invisible boundary at settlement edge
- [ ] Different ambient audio zone inside vs outside settlement
- [ ] "Entering [Name]" toast notification on crossing boundary

### SI-5 — Level of Detail
- [ ] Settlement visible from 80u as simple billboard cluster
- [ ] Full 3D geometry within 40u
- [ ] NPCs only spawned within 20u of player

### SI-6 — Tests
- [ ] Same realm seed → same settlement positions every run
- [ ] All 3 settlement sizes (village/town/city) spawn without Three.js errors

## Dependencies
- Requires: PROC-B `buildBuilding()` + `buildNpc()` 🔲
- Requires: RI-1 terrain mesh (need ground height)
- Feeds: quest givers (05-content), NPC interactions
